# COLOR FLIPPER APP 

A fully interactive application that generates random hex colors, applies them to the background, copies them to the clipboard, and saves the color history to localStorage.

## FEATURES

- **Random Color Generation** – Click the "Flip" button to generate a random hex color
- **Display Color Code** – Shows the hex code of the current color on screen
- **Copy to Clipboard** – "Copy" button copies the color code to your clipboard
- **Toast Notification** – Confirmation message appears when a color is copied
- **Color History** – Stores the last 6 colors used
- **Restore Colors** – Click any color from history to apply it again
- **Persistent Storage** – Color history is saved to localStorage and survives page refreshes

## TECHNOLOGIES

- HTML5
- CSS3 (Flexbox/Grid, animations)
- JavaScript (ES6+)
- Vercel (hosting)
